Connect Blackboard

  1. In a new tab, go to chrome://extensions/. At the top right, turn on Developer mode.
  2. Download the extension zip file
  3. Unzip this file (Go to file explorer and find the file. Right click it then click "Extract All")
  4. On the extensions page (chrome://extensions/), at the top left, click "Load Unpacked". Now select the folder you extracted.
  5. Click the extension icon in the top right of your browser. If you don't see it, click the puzzle icon next to the URL bar to see all extensions. Then click the StudentMate Bridge extension.
  6. Type in your TigerID username and click save.
  7. Go to Blackboard, login, and reload the page.
  8. Finally, reload this page.
